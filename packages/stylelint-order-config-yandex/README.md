# stylelint-order-config-yandex

## Install

```sh
npm i -D stylelint stylelint-order stylelint-order-config-yandex
```

## Usage

```json
{
  "stylelint": {
    "extends": [
      // ...
      "stylelint-order-config-yandex"
      // ...
    ],
    "plugins": [
      // ...
      "stylelint-order"
      // ...
    ]
  }
}
```
