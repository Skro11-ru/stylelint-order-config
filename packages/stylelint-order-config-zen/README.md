# stylelint-order-config-zen

## Install

```sh
npm i -D stylelint stylelint-order stylelint-order-config-zen
```

## Usage

```json
{
  "stylelint": {
    "extends": [
      // ...
      "stylelint-order-config-zen"
      // ...
    ],
    "plugins": [
      // ...
      "stylelint-order"
      // ...
    ]
  }
}
```
