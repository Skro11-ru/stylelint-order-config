# stylelint-order-config-csscomb

## Install

```sh
npm i -D stylelint stylelint-order stylelint-order-config-csscomb
```

## Usage

```json
{
  "stylelint": {
    "extends": [
      // ...
      "stylelint-order-config-csscomb"
      // ...
    ],
    "plugins": [
      // ...
      "stylelint-order"
      // ...
    ]
  }
}
```
