# stylelint-order-config

## Install

```sh
npm i -D stylelint stylelint-order stylelint-order-config-standard
```

## Usage

```json
{
  "stylelint": {
    "extends": [
      // ...
      "stylelint-order-config-standard"
      // ...
    ],
    "plugins": [
      // ...
      "stylelint-order"
      // ...
    ]
  }
}
```
